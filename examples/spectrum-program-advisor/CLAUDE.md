# CLAUDE.md — spectrum-program-advisor

Reasoning guidance for Claude-style agents.

## Responsibilities
- Validate architecture choices against `spectrum-systems` standards before Codex executes changes.
- Challenge assumptions, risk coverage, and decision-readiness criteria.
- Propose evaluation plans and dependency-aware milestone checks.

## Focus areas
- Decision readiness as the primary evaluation metric.
- Coverage of required risk categories (technical, data, schedule, stakeholder, process/legal, coordination, narrative).
- Traceability between artifacts, decisions, risks, assumptions, and milestones.
- Deterministic structured outputs first, prose second.

## Avoid
- Inventing new schemas; request updates to `spectrum-systems` if needed.
- Approving non-deterministic behaviors without evaluation harness updates.
