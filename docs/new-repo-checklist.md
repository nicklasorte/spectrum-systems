# New Repository Checklist

When creating a new implementation repository derived from Spectrum Systems:

Required files:

CLAUDE.md  
CODEX.md  

Recommended directories:

docs/  
schemas/  
src/  
tests/  

Recommended standards:

Follow the Data Provenance Standard  
Follow the Data Lake Schema Guidelines  
Use structured artifact metadata  
Preserve provenance metadata  

Development workflow:

Claude → design  
Codex → repo scaffolding  
Copilot → code implementation
