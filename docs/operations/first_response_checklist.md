# First Response Checklist

1. Identify latest failing `batch_id`.
2. Retrieve corresponding `batch_handoff_bundle`.
3. Inspect `drift_detection_record`.
4. Inspect `capability_readiness_record`.
5. Inspect `decision_proof_record` (and linked control decision).
6. Inspect promotion/policy/override artifacts.
7. Determine whether halt behavior is correct fail-closed behavior.
