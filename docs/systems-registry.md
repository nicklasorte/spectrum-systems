# Systems Registry (Compatibility Companion)

This file is a compatibility companion for readers/tools that still look for `docs/systems-registry.md`.

## Authority and scope
- Canonical subsystem ownership authority: `docs/architecture/system_registry.md`.
- Companion ecosystem registry summary: `docs/system-registry.md`.
- Machine-readable registry: `ecosystem/system-registry.json`.

This document does not define ownership authority and must not be treated as a second registry constitution.

## Current use
- Legacy pointer for SYS-00x engine inventory navigation.
- Bridge for historical links that previously treated this file as canonical.

## Migration guidance
When possible, update links to:
1. `docs/architecture/system_registry.md` for ownership, acronyms, and placeholder status.
2. `docs/system-registry.md` for concise ecosystem repository inventory.
3. `SYSTEMS.md` and `docs/system-map.md` for operational engine asset mapping.
