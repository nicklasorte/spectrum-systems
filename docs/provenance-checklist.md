# Provenance Checklist

## Record Identification
- [ ] Does the artifact have a stable ID?
- [ ] Is the record type defined?
- [ ] Is the title or description clear?

## Source Traceability
- [ ] Is the source document identified?
- [ ] Is the source location identified?
- [ ] Is the source revision captured?

## Generation Traceability
- [ ] Is the generating workflow named?
- [ ] Is the generating system recorded?
- [ ] Are parent entities linked?

## Reviewability
- [ ] Is review status present?
- [ ] Is reviewer identity captured?
- [ ] Is review date present?

## Versioning
- [ ] Is version number present?
- [ ] Is derivation captured?
- [ ] Are superseded versions retained?

## Confidence and Quality
- [ ] Is confidence score present?
- [ ] Is quality status present?
- [ ] Are usage constraints or caveats recorded?
