# Repository Map

## Purpose
Provide a quick orientation for humans and AI agents navigating the repository.

## Top-Level Structure
- README.md — High-level description of the repository.
- SYSTEMS.md — Central index of automation systems.
- CLAUDE.md — Instructions for reasoning agents.
- CODEX.md — Instructions for repository execution agents.
- docs/ — Architecture and design documentation.
- schemas/ — Structured schema definitions.
- issues/ — Research questions and backlog items.
- workflows/ — Conceptual workflow descriptions.
- examples/ — Illustrative artifacts.
