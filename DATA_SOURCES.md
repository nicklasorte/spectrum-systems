# Data Sources

## Purpose
Defines the types of sources that populate the spectrum systems data lake.

## Source Categories
- **Primary Sources**: Original engineering documents or datasets.
- **Derived Sources**: Structured artifacts derived from primary sources.
- **Reference Sources**: External documents used for contextual knowledge.

## Example Sources
- working papers
- comment spreadsheets
- meeting transcripts
- engineering simulations
- allocation tables

Each source should include metadata describing:
- source ID
- origin
- revision history
- trust level
